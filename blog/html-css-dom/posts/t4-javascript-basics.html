Explain control flow and loops using an example process from everyday life, for example 'waking up' or 'brushing your teeth'.

Control flow is the order in which we do things like wake up, then get out of bed, then get dressed, but we might have breakfast before we get dressed on some days and this is still controlled flow. Loops are like habit cycles we do, say when we brush our teeth, then we floss and then mouthwash, this is a loop we do every day in the same order.
